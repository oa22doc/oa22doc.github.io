Summary of API Changes

Following is a summary of the oaVerilog API changes between the OpenAccess p034 release and the p028 release.

#################################
# The following classes were modified #
#################################

	CallbacksIn 


		Added API

		fitNet(oaModNet *net, const oaName &termName, oaModInst *inst)

		widthWarning(const oaModNet *net, const oaName &termName, oaModInst *inst) 

		widthWarning(const oaModNet *net, const oaModTerm *term, oaModInst *inst)


		Removed API

		fitNet(oaModNet *net, const oaName &termName)

		widthWarning(const oaModNet *net, const oaName &termName)

		widthWarning(const oaModNet *net, const oaModTerm *term)

	Error 


		Added API

		Error(oaString fileNameIn, oaUInt4 lineNumIn, oaUInt4 msgIdIn,...) 

		getFileName() const 

		getLineNumber() const 

		hasFileAndLineInfo() const

#################################
