LEF 5.4 Constructs

The lef2gn translator parses all LEF 5.4 constructs. Constructs that are not supported in this current release are ignored and the translator issues a message stating "not yet supported".

Obsolete constructs are those which were supported in LEF/DEF 5.3 or earlier versions and are now obsolete in LEF/DEF 5.4.

Constructs that will not be supported are those that support gate array style designs. OpenAccess 2 supports custom and automated design styles, analog and digital and mixed-signal designs.

Please use the following legend to determine the status of LEF 5.4 constructs in OpenAccess 2.0.
feature -- The feature is implemented
feature -- The feature will be implemented
feature -- The feature is obsolete
feature -- The feature will not be supported
