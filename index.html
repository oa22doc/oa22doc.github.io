OpenAccess C++ API Documentation

The OpenAccess API is a C++ programming interface for integrating design tools and design methodologies through a common, open database. OpenAccess documentation consists of several types, which are listed below. You can access these different types of documentation from this page or by using the navigation bar at the top of every page. For additional information about using the documentation, click the Help icon on the navigation bar.
