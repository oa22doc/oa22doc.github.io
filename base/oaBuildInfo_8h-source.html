00005 // This file contains the definition of the oaBuildInfo class. The oaBuildInfo
00006 // class implements a utility class for registering and querying build
00007 // information for libraries and executables.
00008 //
00009 // The following information is currently stored in the buildInfo:
00010 //
00011 //  packageName     - The name of the library or executable
00012 //  buildName       - The build name, e.g., "a014"
00013 //  buildTime       - The build time as a string, e.g., "Sun Jun  6 12:08 2004"
00014 //  buildNumber     - The build number, e.g., 14
00015 //  majorReleaseNum - The major release number, e.g., 2
00016 //  minorReleaseNum - The minor release number, e.g., 2
00017 //
00018 // The oaBuildInfoArray class implements a utility array class used to pass
00019 // an array of oaBuildInfo pointers back to the user via
