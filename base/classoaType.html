Detailed Description

The oaType class is an enum wrapper for the values that indicate the type of any managed database object. All persistent design database, technology database, and library database objects are included.

Abstract base classes are not included in the list of types. Use the isClass functions on various base classes to classify an object by one of the base classes. For example, you can use isShape() on any oaBlockObject.

See oaObject::getType for its usage. See Enum Wrappers in the Programmers Guide for a discussion of enum wrappers.
