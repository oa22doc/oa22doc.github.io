DEF 5.4 Constructs

The def2oa translator is a DEF Reader that parses all DEF 5.4 constructs. Constructs that are not supported in this current release are ignored and the translator issues a message stating "not yet supported".

Obsolete constructs are those which were supported in LEF/DEF 5.3 or earlier versions and are now obsolete in LEF/DEF 5.4.

Constructs that will not be supported are those that support gate array style designs. OpenAccess 2 supports custom and automated design styles, analog and digital and mixed-signal designs.

Please use the following legend to determine the status of DEF 5.4 constructs in the OpenAccess 2.0 DEF Reader.
feature in black -- The feature is implemented
feature in red -- The feature will be implemented in the Community Release in December
feature in purple -- The feature is obsolete in DEF 5.4 and will not be implemented
feature in blue -- The feature is not supported in OpenAccess

All 5.5 constructs are parsed. Features that are not supported TODAY are ignored and skipped.
