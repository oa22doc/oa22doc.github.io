Extension and Property Class Index

Several OpenAccess classes allow applications to extend databases to support their own private data model. One common way to extend the database is by using OpenAccess properties and groups. Another way is to use the OpenAccess extension classes to create application-defined database extensions.

See oaProp Class Reference, oaGroup Class Reference, and Extending the Database for more information.
