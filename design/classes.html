Design Class Index

An design database holds an integrated representation of all the data related to a particular element in a design. This includes the design’s connectivity, geometry, hierarchy, parasitic, and floorplanning information. This category includes all the classes that can be stored in oaDesign databases and the classes that are used to interact with those objects.

See Understanding Logical Connectivity, Understanding Routes, and Understanding OpenAccess Hierarchy in the Programmers Guide for more information.
