Detailed Description

The oaSource class encapsulates the enumerated values that describe the different sources you can assign to an instance or a net.

enum oaSourceEnum
 

Enumeration values:
oacNetlistSource  "netlist"
oacTimingSource  "timing"
oacDistSource  "dist"
oacTestSource  "test"
oacUserSource  "user"
