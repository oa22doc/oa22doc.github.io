Detailed Description

The oaSymmetry class encapsulates the enumerated values that describe the different symmetries of an oaDesign object.

enum oaSymmetryEnum
 

Enumeration values:
oacNoSymmetry  "none"
oacXSymmetry  "x"
oacYSymmetry  "y"
oacXYSymmetry  "xy"
oacAnySymmetry  "any"
