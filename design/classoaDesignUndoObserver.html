template<oaUInt4 rev = oacObserverVersion2>
class oaDesignUndoObserver< rev >

This class allows for the construction of an observer for actions associated with undo and redo functions.

This function constructs an oaDesignUndoObserver callback object. If multiple observers exist for a given design undo notification event, observers with a lower priorityIn value are called before those with a higher value.

Parameters:
priorityIn priority of this observer relative to other design undo observers
enabledIn Determines whether this observer is called when an observed event occurs
