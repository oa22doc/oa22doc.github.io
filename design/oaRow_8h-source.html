00005 // This file contains the definition for the oaRow class. The oaRow class
00006 // implements a row of sites at which a block may be placed. A row 
00007 // represents a potential location for placement of standard cells, macros, or
00008 // devices. A row is defined as an area associated with a oaSiteDef. The
00009 // oaSiteDef has information allowing for the  specification of the placeable
00010 // objects within a row. A row is a repetition of a site element in a given 
00011 // direction (horizontal or vertical).
