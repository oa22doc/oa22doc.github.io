00005 // This file contains the definition for the oaModModuleVectorInstBit class. The
00006 // oaModModuleVectorInstBit class implements a single bit of a module vectorInst
00007 // in a module. VectorInstBits are automatically created and destroyed as
00008 // vectorInsts are created or destroyed. Implicitly created vectorInstBits
00009 // cannot be modified to insure consistency with its defining vectorInst.
00010 // Explicitly created vectorInstBits define a unique bit of a vectorInst
00011 // definition since the bits of a vectorInst are not allowed to overlap with
00012 // other vectorInsts.
