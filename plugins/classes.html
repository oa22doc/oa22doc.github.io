Plug-In Class Index

OpenAccess allows certain functions of the API to be customized at run time through the use of plug-ins. Plug-ins let the end user choose the implementation for select capabilities using software that may not have been available when the application was linked.

See How to Write a Plug-In, Writing a C++ Pcell Generator, How to Write a Pcell Evaluator Plug-In, Implementing Pcells Through Tcl, Region Query Plug-In Architecture, How to Write Change Management System (CMS) Plug-Ins, and How to Write a Plug-in to Calculate Bounding Boxes for Text for more information.
