OpenAccess Change Management System (CMS)

OpenAccess CMS defines a plug-in based infrastructure that lets software developers create plug-ins for incremental data exchange between cooperating applications.

NOTE: All documentation for CMS is preliminary and subject to change. CMS is still in development and is considered to be of Beta quality.

See Overview of Change Management System (CMS) Classes, Change Management System (CMS), How to Write Change Management System (CMS) Plug-Ins, Creating Schema Definitions, and Traversing and Iterating Schema Definition Objects for more information.
