How to Write Change Management System (CMS) Plug-Ins

The OpenAccess CMS infrastructure provides a standard way for an application to select and configure the plug-ins used to exchange data between applications.

For more information, refer to the following documents:

  • How to Write a CMS Tracking Plug-In
  • How to Write a CMS Export Plug-In
  • Importing Changes from the Change Management System (CMS)

For an overview of the CMS system, refer to Change Management System (CMS) in the Programmers Guide.

Important: All documentation for CMS is preliminary and subject to change. CMS is still in development and is considered to be of Beta quality.
