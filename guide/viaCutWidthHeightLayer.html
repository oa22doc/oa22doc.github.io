Description

The width, height, and layer via parameters determine the size of each cut rectangle and the layer number for this cut. By default, the origin (the 0,0 point of the coordinates) is at the center of the cut rectangle. For vias with multiple cut rectangles the default origin is in the center of the bottom left cut rectangle. See Origin Offset to move this default.

  • setCutWidth specifies the width of the cut, as an integer (in DBUs).


  • setCutHeight specifies the height of the cut, as an integer (in DBUs).


  • setCutLayer specifies the layer number of the cut layer.

Use the corresponding get functions to return this information from the database.
