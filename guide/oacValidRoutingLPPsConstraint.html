validRoutingLPPs (oacValidRoutingLPPs)

Description

This constraintDef specifies a list of valid layer-purpose-pairs to be used in routing. It is intended to be used in a purpose-specific layerArray constraint. This implies that the constraint is created with an array of layers and an array of purposes.

The number of layers of this layerArrayConstraintDef is zero, indicating that any number of layers can be used when creating a constraint with this constraintDef.

Constraint at a Glance

Constraint type: oaLayerArrayConstraint (Number of layers: 0)
Value types: oaBooleanValue
Database types: oaDesign, oaTech
Object types: oaRoute, oaScalarNet, oaBusNetBit, oaAppObject

Values

The following value types are supported by this constraint:

  • oaBooleanValue

    This value is not used and should be "true". The allowed layer-purpose pairs are specified as oaLayerArrays and oaPurposeArrays when creating the constraint.

    Units: Boolean
