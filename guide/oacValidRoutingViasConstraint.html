validRoutingVias (oacValidRoutingVias)

Description

The built-in simple constraint definition oacValidRoutingVias specifies the vias that can be used in routing. The constraint can include the actual specifications of particular vias, a list of the via definitions, or a combination of the two. In addition, the constraint can specify valid or default power routing vias.

Constraint at a Glance

Constraint type: oaSimpleConstraint
Value types: oaViaDefArrayValue, oaViaTopologyArrayValue, oaViaTopology2DTblValue
Database types: oaDesign, oaTech
Object types: oaRoute, oaScalarNet, oaBusNetBit, oaAppObject

Values

The following value types are supported by this constraint:

  • oaViaDefArrayValue

    This oaViaDefArrayValue is considered deprecated and is supported for compatibility purposes.

    Units: Via Def

  • oaViaTopologyArrayValue

    This oaViaTopologyArrayValue contains a set of via specifications(oaViaVariants), a list of valid vias(oaViaDefs), or a combination of the two.

    Units: Via Topology

  • oaViaTopology2DTblValue

    This oaViaTopology2DTblValue can be used to provide valid or default power routing vias.

    Units: Via Topology
