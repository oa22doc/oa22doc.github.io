minGateExtension (oacMinGateExtension)

Description

This constraint specifies the gate extension if a layer1 (poly) shape is within specific distance of an L-shape of the same MOS layer2 (oxide) shape.

Constraint at a Glance

Constraint type: oaLayerPairConstraint (Symmetric: no)
Value types: oaIntValue
Database types: oaDesign, oaTech
Object types: oaAppObject

Values

The following value types are supported by this constraint:

  • oaIntValue

    This value is the required minimum extension.

    Units: DBU

Parameters

The following parameters are supported by this constraint:

Name Value Type Units Default Description
distance
oacDistanceConstraintParamType		 oaIntValue DBU None

This parameter holds the spacing value. The constraint would only apply if the spacing between the two layers is < than the parameter value.
minJogLength
oacMinJogLengthConstraintParamType		 oaIntValue DBU None

This parameter holds the value of the maximum allowed jog up until which the rule does not apply. The rule would not apply if the oxide jog is =< this parameter value.

Examples


Manufacturing Reasons

This constraint ensures sufficient poly overhang past gate forming connections.
