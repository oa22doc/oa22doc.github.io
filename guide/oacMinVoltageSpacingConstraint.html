minVoltageSpacing (oacMinVoltageSpacing)

Description

The minimum spacing required between two wire shapes on the same metal layer which have different voltages where the voltage is greater than or equal to the given voltage.

Constraint at a Glance

Constraint type: oaLayerConstraint
Value types: oaValueArrayValue
Database types: oaDesign, oaTech
Object types: oaAppObject

Values

The following value types are supported by this constraint:

  • oaValueArrayValue

    The constraint value consists of float-Int pairs where voltage is represented as a float and the spacing value is represented as an integer in database units. The voltage index represents the maximum voltage difference between the two shapes. The voltage value would have a >= semantic, that specifies the required clearance which applies to all voltage differences greater than or equal to it. Refer to the example in minVoltageClearance for a detailed description.

    Units: DBU
