oaDefaultVerticalRouteGridOffset (oacDefaultVerticalRouteGridOffset)

Description

The built-in oacDefaultVerticalRouteGridOffset constraint is an oaSimpleConstraint type that specifies the default vertical route grid offset for all layers. The offset distance is specified in database units. This constraint applies if the oacVerticalRouteGridOffset is not set.

Constraint at a Glance

Constraint type: oaSimpleConstraint
Value types: oaIntValue
Database types: oaDesign, oaTech
Object types: oaAppObject

Values

The following value types are supported by this constraint:

  • oaIntValue

    This oaIntValue represents the default route grid offset in database units.

    Units: DBU
