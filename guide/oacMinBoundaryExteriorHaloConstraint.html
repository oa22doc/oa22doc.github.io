minBoundaryExteriorHalo (oacMinBoundaryExteriorHalo)

Description

This constraint defines the minimum spacing between shapes on the specified layer in the current level of hierarchy and the PRBoundaries of all instances at the current level of hierarchy.

Constraint at a Glance

Constraint type: oaLayerConstraint
Value types: oaIntValue
Database types: oaDesign, oaTech
Object types: oaPRBoundary, oaAppObject

Values

The following value types are supported by this constraint:

  • oaIntValue

    This value defines the required spacing to the PRBoundary.

    Units: DBU

Parameters

The following parameters are supported by this constraint:

Name Value Type Units Default Description
coincidentAllowed
oacCoincidentAllowedParamType		 oaBooleanValue Boolean False

When this Boolean parameter is set to true, it specifies that shapes can either meet the minimum distance specified or their edges can be coincident with the boundary. If the parameter is absent or is false, the edges must meet the minimum distance specified and cannot be coincident.
