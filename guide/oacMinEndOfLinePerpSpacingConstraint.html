minEndOfLinePerpSpacing (oacMinEndOfLinePerpSpacing)

Description

Some processes require a particular end of line spacing for lines with a width less than a given width beyond the end-of-line to a perpendicular edge with width less than a minimum width. Typically, spacing is slightly larger than the minimum allowed regular spacing or end of line spacing on the layer. Although, only one such end of line spacing rule is expected for a given layer, there can be an end of line spacing rule to a perpendicular edge rule coupled with multiple regular spacing or end of line spacing rules for a single layer.

Constraint at a Glance

Constraint type: oaLayerConstraint
Value types: oaIntValue
Database types: oaDesign, oaTech
Object types: oaAppObject

Values

The following value types are supported by this constraint:

  • oaIntValue

    This value specifies the end of line spacing for lines with a width less than a given width.

    Units: DBU

Parameters

The following parameters are supported by this constraint:

oacWidthConstraintParamType		 oaIntValue DBU (Required)

This parameter stores the minimum width in database units.

endOfLinePerpWidth
oacEndOfLinePerpWidthConstraintParamType		 oaIntValue DBU (Required)
