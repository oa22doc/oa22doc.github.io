Constraint and Value Class Index

The OpenAccess oaConstraint class provides a base class for constraint representations. All constraint objects are defined by the constraintDef. Constraints have a value associated with them and potentially a list of constraint parameters.

See Creating and Modeling Process Rules and Constraints for more information.
