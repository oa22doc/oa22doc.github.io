Block Domain Class Index

Block objects specify the physical aspects of a design. These include the block domain connectivity and all geometric aspects of the design, which are represented by the classes derived from oaFig. This category includes all the classes that can be stored in an oaBlock and the classes that are used to describe those objects.

See Physical Routing Segments and Using Transforms in the Programmers Guide for more information.
