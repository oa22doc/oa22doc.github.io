SPEF Translators

The SPEF translators consist of spef2oa for reading parasitic information in a SPEF file to annotate an OpenAccess database and oa2spef for reading parasitic information stored in an OpenAccess database to create a SPEF file. For information about using the SPEF translators, see:

  • spef2oa translates SPEF information and annotates an OpenAccess database.
  • oa2spef translates the parasitic information stored in an OpenAccess database and creates a SPEF file.

See SPEF to OpenAccess Mapping for information about how the SPEF syntax maps to OpenAccess objects.
